# AWS Lightsail

O **AWS Lightsail** é um serviço da AWS voltado para **iniciantes em cloud** e para quem quer **subir aplicações completas de forma simples e rápida**.

Ele abstrai grande parte da complexidade da AWS, entregando uma experiência mais direta.

---

## O que o Lightsail Entrega

O Lightsail permite subir, de forma simplificada:
- Servidores (EC2)
- Balanceamento de carga
- Armazenamento
- Banco de dados
- Aplicações prontas (WordPress, Node.js, etc.)

Tudo isso com **configuração mínima**.

---

## Características Principais

- Interface simples
- Preços previsíveis (pacotes mensais)
- Ideal para pequenos projetos
- Menos flexível que EC2 puro, porém muito mais fácil de usar

---

## Casos de Uso Comuns

- Projetos pessoais
- MVPs
- Pequenos sites e APIs
- Ambientes de teste
- Quem está começando na AWS

---

## Quando Usar AWS Lightsail

- Quando você quer **simplicidade**
- Quando não precisa de configurações avançadas de rede e segurança
- Quando quer subir tudo rápido sem se preocupar com a arquitetura interna da AWS
