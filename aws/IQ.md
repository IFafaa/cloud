# AWS IQ

O **AWS IQ** é uma **plataforma de freelancers gerenciada pela AWS**, focada em conectar **clientes** a **especialistas certificados em AWS**.

Ela facilita a contratação de profissionais qualificados para projetos em cloud.

---

## Objetivo do AWS IQ

- Conectar empresas a especialistas AWS
- Garantir qualidade técnica
- Facilitar contratações de curto ou médio prazo
- Acelerar projetos em cloud

---

## Como Funciona

1. O cliente publica uma necessidade (ex: arquitetura, migração, segurança)
2. Especialistas certificados respondem à solicitação
3. O cliente escolhe o profissional
4. O projeto é executado com suporte da AWS

---

## Tipos de Serviços Oferecidos

- Arquitetura de soluções
- Migração para AWS
- Otimização de custos
- Segurança e compliance
- Revisão de arquitetura
- Troubleshooting

---

## Vantagens do AWS IQ

- Profissionais **verificados e certificados**
- Plataforma oficial da AWS
- Menos risco na contratação
- Foco em especialistas cloud

---

## Quando Usar AWS IQ

- Quando não há expertise interna em AWS
- Para projetos específicos ou pontuais
- Para acelerar entregas críticas
- Para revisões técnicas especializadas

---

## Resumo

O AWS IQ é a **ponte entre empresas e especialistas AWS**, oferecendo segurança, qualidade e agilidade na contratação.
