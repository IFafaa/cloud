# AWS Fargate

O **AWS Fargate** é um serviço de **execução de containers serverless**.

Você roda containers (Docker) **sem precisar gerenciar servidores ou clusters**.

---

## Em Resumo

- Executa containers
- Não gerencia EC2
- Paga só pelo uso
- Ideal para ECS ou EKS

---

## Quando Usar AWS Fargate

- Quando quer rodar containers sem se preocupar com infraestrutura
- Quando quer foco total no código e no container
