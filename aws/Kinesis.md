# AWS Kinesis

O **AWS Kinesis** é o serviço da AWS voltado para **processamento de dados em tempo real (real-time data)**.

Ele permite:
- Captura de dados
- Processamento / analytics
- Armazenamento de streams de dados

Tudo isso acontecendo **em tempo real**.

---

## Características Principais

- Trabalha com **streams de dados contínuos**
- Ideal para grandes volumes de eventos
- Processamento com baixa latência
- Escalável automaticamente

---

## Tipos de Dados Capturados

- Eventos de aplicações
- Logs
- Métricas
- Dados de cliques
- Dados de sensores / IoT

---

## Casos de Uso Comuns

- Quantidade de usuários ativos em uma live / stream
- Métricas em tempo real (ex: acessos por segundo)
- Monitoramento de aplicações
- Análise de comportamento de usuários
- Processamento de logs em tempo real

---

## Exemplos Práticos

- Contar quantas pessoas estão assistindo uma live agora
- Detectar picos de acesso em tempo real
- Coletar eventos de aplicações e gerar dashboards
- Analisar dados conforme eles chegam, sem precisar esperar processamento em batch

---

## Quando Usar Kinesis

- Quando **tempo real é crítico**
- Quando há **alto volume de eventos**
- Quando não é aceitável esperar processamento posterior
